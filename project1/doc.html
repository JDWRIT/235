Site goal:

To act as a showcase of my A.A.R.D. V-ARK project. The audience for the site is employers who would be looking for something I've done to see if I am a good fit for their company. Currently we are still in the "make a site based off a template" so the design decisions have not been my own so far for the look of the website.

In phase 2, I swapped out the content with my own, as well as implementing media queries to make sure the website is still readable at shorter widths. One key feature of this is the change from 3 to 2 columns for the main text when the website gets narrow enough, that way text isn't incredibly smushed on a mobile device. Additionally, I changed the colors of the various states of links and added navigational elements to take you up and down the page without the need to scroll.
